## Description
This program is inspired by the Dutch painter Piet Mondrian. Specifically, his
artwork Composition II in Red, Blue, and Yellow. The painting can be described
as a grid comprised of vertical and horizontal straight lines. These lines
create rectangles, and some of the shapes are colored in with usually vibrant
colors.

The window is divided into two major parts: The left side is for the user to
input their design choices. On the right side is where the canvas is located
and will be presented when the user clicks the Create Painting button.

The design choices that are presented are the number of lines, both vertically
and horizontally, limited to three. The user can also choose the number of
colored rectangles in the painting. One of the more personal options the user
has is to add their signature to the painting, customizing it with two typefaces
to select from and various font sizes.

When the user is happy with their design choices, they can click the Create
Painting button and watch the canvas come to life as it paints their artwork.
The animation begins by setting down the lines randomly to create the grid,
painting the rectangles with randomized colors and finishing off by writing
their signature.

Areas for improvement include limiting the choice of painted rectangles based
on the current values in the vertical and horizontal options. Regarding the
signature, it would be better if you could choose the placement on the canvas,
include more typefaces, and color options. Another option apart from the
rectangle colors being random would be color schemes such as monochromatic,
analogous, complementary, etc.
